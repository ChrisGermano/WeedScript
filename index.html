WeedScript
A[n in-progress] Turing complete language

WeedScript is a basic programming language based off of Brainfuck and interpreted in Java.

The java interpreter looks for a given ".ws" file, at least Java 1.6 is required.



Following several well-known smoking "rules", WeedScript operates in the following manner:


-The "sesh" begins with a circle of X people with value 0, the first person holds the bowl

-The bowl can move (only to the left!) one index at a time.

-People are able to perform actions on the bowl, which in turn can affect their values.

Elements

Pass

Move the bowl to person to the "left" of the current person.

Blaze

Add the current bowl value to the current person.

Pack

Add 1 from the bowl (maximum 231-1).

Cherry

Subtract 1 from the bowl (cannot be negative).

Chill

Subtract 1 from the current person (cannot be negative).

Story

Print out the current person's value.

Toke

Sets the current person to the ASCII value of 1 char of user input.

Input that extends past one character is ignored after the first character.

Sesh

Sets the current sesh to the first integer of user input.

NOTE: This sets the value of all people and the bowl to 0, effectively wiping all existing data.

Ash

Sets the bowl value to 0.

This is shorthand for "inhale cherry exhale"

Inhale

Begins a loop sequence.

All commands between "inhale" and the following "exhale" are repeated

until the bowl or current person's value is 0.

NOTE: Nested loops are in progress and do not currently work

Exhale

Checks loop end conditions.

This checks if the bowl or current person's value is 0,

and if so it terminates the loop, otherwise the loop continues.

Examples

User Operation

-The user begins with interpreter.java and the .ws file in the same directory

-First, they compile interpreter.java with the command "javac interpreter.java"

-Following successful compilation, they run the interpreter on the file with the command "java interpreter [filename].ws"


NOTE: In the future, this will be streamlined and automated

Basic WeedScript

File contents:

pack pack pack pack inhale blaze cherry exhale story


Operation begins with four "pack" commands, increasing the bowl value to 4

The next command begins a loop, and the interpreter looks for the next instance of "exhale"

All commands within "inhale" and "exhale" are stored, in this case "blaze cherry"

When the loop terminates, the current person will be 4 and the bowl value will be 0

Finally, the program will output the character with ASCII value 4 (EOT character) and terminate

Advanced Weeding

File contents:

sesh pack pack pack pack pack pack pack

pack pack pack inhale blaze cherry exhale

chill chill chill story chill chill story

chill chill story


Operation begins with a "sesh" initializer, where the user must input any positive integer.

NOTE: Input value must be less than 231-1.

The code then packs the bowl ten times, and begins a loop. The loop contains two commands,

"blaze cherry", which add the current bowl value to the current person and decrement it by 1,

which results in the current person value being 55 when the loop terminates. The current person

value is then decremented by 3 with the "chill" commands, and its value (52) is printed,

resulting in a "4". The bowl value is then decremented twice and printed two more times,

finally ending with a complete program output of "420".

Lazy Weeding

File contents:

toke story toke story toke story


The user is prompted for three inputs, where they pass in 52, 50, and 48 in that order.

The program outputs "420"
